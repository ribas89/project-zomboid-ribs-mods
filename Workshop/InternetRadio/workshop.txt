description=[h2]What this mod does[/h2]
description=This mod brings [b]real-world internet radio stations[/b] into Project Zomboid. You can tune in to live streams directly in-game using custom frequencies.  
description=It also adds options to configure:  
description=- [b]minimum and maximum frequencies[/b] available on radios  
description=- [b]minimum and maximum distance[/b] the radio sound can be heard
description=
description=[h2]How to add new stations[/h2]
description=There are two ways to add your own stations:
description=
description=[h3]1. Through Mod Options / Sandbox[/h3]
description=- Open Mod Options or Sandbox settings.
description=- Add your custom station entries there.
description=
description=[h3]2. Through Lua modding[/h3]
description=[h2]Video tutorial on how to add stations:[/h2]
description=https://www.youtube.com/watch?v=LRSQTiiZ7nA
description=
description=Use the Lua command:  
description=[code]InternetRadio:addStation(frequency, name, url)[/code]  
description=
description=Example:  
description=[code]InternetRadio:addStation(102.6, "Word of the LORD", "https://listen.radioking.com/radio/299755/stream/346354")[/code]  
description=
description=To add custom categories, use a fourth parameter:  
description=[code]InternetRadio:addStation(frequency, name, url, category)[/code]  
description=
description=Example:  
description=[code]InternetRadio:addStation(102.6, "Word of the LORD", "https://listen.radioking.com/radio/299755/stream/346354", "Christian")[/code]  
description=

description=[h2]Important about stations[/h2]
description=- [b]Not every internet radio will work[/b]. Stations that require you to press "Play" on a website usually won’t function in-game.  
description=- Stations that autoplay when opened in a browser have the best chance of working.  
description=
description=[h2]Important about frequencies[/h2]
description=Project Zomboid radios only work on certain steps:
description=- Frequencies increase in increments of [b]0.2[/b] (for example: 95.0 ? 95.2 ? 95.4).  
description=- This means you cannot set arbitrary numbers like [b]95.1[/b] or [b]95.3[/b].  
description=- If you set an odd frequency, it will automatically try to adjust to the nearest valid even step.  
description=- In case two stations use the same frequency, the last one added will automatically shift up by [b]+0.2[/b].  
description=
description=To help you quickly find valid stations, it is highly recommended to use this mod:  
description=[url=https://steamcommunity.com/sharedfiles/filedetails/?id=3566943255]Easy Frequency Preset[/url]  
description=
