title=Custom Zoom Parameter
description=[b]Makes camera zoom levels sandbox configurable[/b] via Custom Zoom Parameter.
description=This mod ignores the vanilla zoom options and uses the values you set in Sandbox / Mod Options.
description=
description=You decide the zoom levels in Mod Options.
description=Examples of what you can do:
description=* Raise the maximum zoom beyond vanilla
description=* Use fine steps like 0.25 for smooth zooming
description=* Keep only a few clean steps like 2.0 1.0 0.5
description=* Create an ultra wide range such as 10.0 down to 0.25
description=
description=[b]How to install[/b]
description=1. Subscribe to the mod
description=2. Open the mod location  
description=   [code]SteamLibrary\steamapps\workshop\content\108600\3562684029\mods\CustomZoomParameter[/code]
description=3. Copy [code]MultiTextureFBO2.class[/code] from the mod
description=4. Go to  
description=[code]SteamLibrary\steamapps\common\ProjectZomboid[/code]
description=5. Enter  
description=[code]zombie\core\textures[/code]
description=6. Paste and replace [code]MultiTextureFBO2.class[/code] there
description=7. Launch the game
description=
description=[b]How to change zoom options[/b]
description=1. Enable the mod in your save
description=2. Load the save
description=3. Go to Mod Options
description=4. Modify the zoom by the mod option instructions
description=Example default:
description=[code]2.25;2.0;1.75;1.5;1.25;1.0;0.75;0.5;0.25[/code]
description=Example full range:
description=[code]12.0;9.0;6.0;5.0;4.0;3.0;2.5;2.25;2.0;1.75;1.5;1.25;1.0;0.75;0.5;0.25;0.15;0.10;0.05;0.025;0.020;0.015;0.01;0.005;0.004;0.003;0.002;0.001[/code]
description=5. Apply the new values
description=6. Scroll Zoom
